# react-native-demo

# 系统环境
```shell
$ react-native -v
react-native-cli: 2.0.1
react-native: n/a - not inside a React Native project directory

$ node -v
v10.11.0

$ npm -v
6.4.1
```
# 安装
1.__安装xcode__

```shell
brew install node
brew install watchman

npm install -g react-native-cli
```



__windows用户查看[官方文档](https://facebook.github.io/react-native/docs/getting-started.html)__

# 创建项目
```shell
react-native init 你的项目名
cd 你的项目名
react-native run-ios / react-native run-android
```
